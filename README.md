# 模式识别与计算机视觉 2025

## 课程概述
本课程介绍模式识别与计算机视觉的基础理论与应用，涵盖图像处理、特征提取、分类和检测算法等内容。同时，通过8次专题课介绍计算机视觉前沿进展，包括研究问题，数据库和前沿算法。本页面用于8次专题课相关材料下载，如有问题请联系 wt [@] smail.nju.edu.cn 。
